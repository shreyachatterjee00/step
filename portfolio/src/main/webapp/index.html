Sentiment Score & Entity Extraction

The Natural Language API consists of Machine Learning models that understand and extract features of sentences. Very interesting stuff for the Linguist in me!

The ML models in the API generate a sentiment score of a sentence from -1 to 1, depending on how positive or negative the message is. Find out the score for your sentence here!







The models can also extract 'entities' of sentences, such as people, organizations, etc. Get a list of entities in your message here!
